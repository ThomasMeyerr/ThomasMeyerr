À propos

Qui suis-je ?

Un développeur qui prend soin des détails — de l'architecture au pixel près.

En quelques mots

Développeur passionné avec une forte sensibilité produit, je conçois des applications mobiles et des systèmes backend avec le même soin que je mets dans les expériences utilisateurs. J'aime les codebases propres, les conventions rigoureuses et les outils qui font gagner du temps. Quand je ne code pas, je réfléchis à comment mieux coder.

Approche

Architecture d'abord, implémentation ensuite. DRY, SOLID, et un sens aigu de quand ne pas sur-abstraire. Le bon code est celui qui se lit comme une phrase.

Centres d'intérêt

Mobile-first, outillage développeur, performance, et l'intersection entre UX et ingénierie. Fan de Rust pour les projets où les nanosecondes comptent.

Actuellement

Focus sur Flutter et les apps mobiles à monétisation intégrée. Exploration des patterns d'agents LLM et de l'automatisation du workflow développeur.

Langues

Français (natif) · Anglais (courant) · Dart · TypeScript · Rust · SQL
